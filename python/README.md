# Python Environment

This directory contains the Python project for Blind 75 interview preparation.

## Setup

1. Activate the virtual environment:
   ```bash
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

After activating the virtual environment, you can run Python scripts and install packages as needed.

To deactivate the virtual environment:
```bash
deactivate
```
